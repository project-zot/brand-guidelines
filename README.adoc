= zot Brand Guidelines

This repository contains Color schemes, logos, and rules for branding. Of the zot project.

== Name

zot is preferably presented all lowercase.  Zot is also acceptable.

== Color Scheme

=== Logo Colors

Primary: #A53692

Secondary: #4A5F12

Tertiary: #F58634 

=== Text Colors

Dark Color: #001E44

Light Color: #FFFFFF
